# VPN Telegram Bot

Бот для автоматической установки и управления VPN-серверами (3X-UI + Reality).

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с токеном бота
4. Запустить: `python main.py`

## Функционал

- Автоматическая установка X-UI на VPS
- Управление через SSH (пароль/ключ)
- Система подписок с оплатой
- Мониторинг серверов
- Админ-панель
